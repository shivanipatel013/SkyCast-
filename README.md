# 🌤 SkyCast – Modern Weather App

A beautiful and responsive Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather data using OpenWeather API.

---

## 🚀 Live Demo

🔗 *Live Project:*  
👉 https://shivanipatel013.github.io/SkyCast-/

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡 Real-time temperature (°C)
- 💧 Humidity information
- 🌈 Dynamic background change based on weather condition
- 📱 Fully responsive design
- ⚡ Smooth UI transitions

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Styling & Animations)
- JavaScript (ES6) 
- Logic & API Handling 
- OpenWeather API
